# CHERWIN FRPC 🚀

> ⚡ 高性能 FRP 内网穿透客户端 (KernelSU / Magisk 专版)

CHERWIN FRPC 是一个专为 Android Root 环境（KernelSU & Magisk）打造的内网穿透模块。它内嵌了最新版的 FRPC 核心（v0.69.1），并附带了一个强大、美观且开箱即用的 WebUI 控制面板，让你在手机上极速部署网络服务。

## ✨ 核心特性

- **🚀 极简部署**：一键刷入，开机自启后台守护进程。
- **🌐 独立 WebUI 面板**：内置轻量级 HTTP 服务 (默认端口 `8099`)，告别繁琐的命令行操作。
- **📊 实时状态监控**：
  - FRP 核心进程存活与 PID 监控。
  - **主服务端连接状态实时侦测**（在线、离线、错误原因分析）。
  - **各代理通道 (Proxy) 独立状态侦测**，直观显示哪个穿透成功，哪个失败。
  - 设备内存占用与连续运行时间统计。
- **📝 配置热重载**：在 WebUI 中直接编辑 `frpc.toml`，一键保存并平滑重启服务。
- **📖 内置全景文档**：控制面板内嵌完整的配置手册（包含基础配置、宝塔 Nginx 反代实战、Store 持久化配置、常见代理模板等），边看边配，小白也能轻松上手。
- **💾 Store 动态持久化**：全面支持 FRP v0.52+ 的 `store` 特性，防重启掉线。

## 📦 安装与使用

### 1. 刷入模块
1. 下载最新的 `CHERWIN_FRPC_magisk_module.zip` 发布包。
2. 打开 **KernelSU** 或 **Magisk** 管理器。
3. 进入“模块”页面，点击“从本地安装”，选择下载的 ZIP 包。
4. 安装完成后，**重启设备**。

### 2. 访问 Web 控制面板
- **快捷方式**：在 KernelSU 的模块列表中，找到 `CHERWIN_FRPC`，点击右侧的**设置齿轮 (或 `< >` ) 图标**，系统将自动唤起浏览器打开面板。
- **手动访问**：打开手机任意浏览器，访问：`http://127.0.0.1:8099`

---

## 🛠️ 面板功能一览

### 状态监控 (Dashboard)
实时掌控 FRP 运行命脉。直观展示主进程是否存活、与远程服务器的握手状态，以及下方列出的每一个 TCP/HTTP/HTTPS 隧道的连通性。

### 参数配置 (Config Editor)
在线编辑 TOML 配置文件。不再需要使用 MT 管理器去深层目录寻找文件。编辑完成后点击“保存并热重启”即可立即生效。

### 运行日志 (Logs)
无需 SSH 连接手机，直接在网页端拉取 `frpc` 核心的运行 Debug 日志和面板守护脚本的日志。自带关键字高亮解析。

### 配置文档手册 (Docs)
你不需要去网上到处找 FRP 的教程，面板顶部内置了全套中文实战手册，代码片段点击即可复制。

## 💡 典型应用场景

1. **📱 手机变身便携网盘**：配合 FRP 的 `store` 插件，将手机 `/sdcard` 目录挂载到公网，随时随地在电脑上读取手机文件。
2. **🌍 局域网服务公网访问**：在手机上运行 Termux / KSUD 等搭建的建站环境、游戏私服，通过配置快速映射到公网域名。
3. **💻 远程 SSH 运维**：通过 TCP 穿透，在任何地方 SSH 登录回你的 Android 手机终端。

---

## ⚠️ 注意事项

1. 本模块内置的 FRPC 核心版本为 `v0.69.1`，配置格式已全面升级为 **TOML**。不再支持旧版的 `ini` 格式，请务必注意语法规范。
2. 若您的 WebUI 无法保存配置，请检查是否有其他模块干扰了 `/data/adb/modules/` 的写入权限。
3. 长期高并发的网络穿透可能会增加手机耗电与发热，请根据实际需求开启。
